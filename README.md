## 🎯 设计哲学

### 三层闭环

```
┌──────────────────────────────────────────────────┐
│  ① 底盘层（学习科学）                            │
│    - 艾宾浩斯遗忘曲线（1→2→4→7→15→30 天间隔）    │
│    - 番茄钟（15min 提醒 / 25min 强制结束）        │
│    - 自适应难度（连对 3 升级，连错 2 降级）        │
├──────────────────────────────────────────────────┤
│  ② 内容层（教材对齐）                            │
│    - 英语：22 主题 · 190+ 单词 · 6 玩法 · 8 对话  │
│    - 数学：24 章节 · 13 题型生成器 · 应用题圈词    │
│    - 语文：41 课 · 501 字 · 14 古诗 · 6 阅读理解  │
├──────────────────────────────────────────────────┤
│  ③ 数据层（家长可见）                            │
│    - 学生：每日报告 · 错题本 · 12 枚成就勋章       │
│    - 家长：KPI · 热力图 · 错因饼图 · 周目标 · PDF │
└──────────────────────────────────────────────────┘
```

### 教育学依据

| 原则 | 实现方式 |
|---|---|
| 间隔复习（Spaced Repetition） | 错题按艾宾浩斯节点重弹，5 次答对自动毕业 |
| 最近发展区（ZPD） | 三档自适应难度，按近期正确率动态调整 |
| 即时反馈 | 答对立刻 ⭐+音效+鼓励语，答错弹 💡 解释 |
| 元认知（Metacognition） | 错答后 chips 让孩子选错因，养成"反思错在哪"习惯 |
| 番茄工作法 | 三年级专注上限 15min，到点提醒，25min 强制休息 |
| 审题训练 | 应用题做题前"圈关键词"前置步骤，分离审题与计算 |
| 家校共育 | 家长后台数据看板 + PNG/A4/JSON 三种导出 |

## 📦 模块速览

### 🇬🇧 英语乐学（译林版）
- **22 主题 / 190+ 词**：3A·U1-U8 + 3B·U1-U8 + 6 个拓展（水果/动物/身体/天气/交通/动作）
- **6 个玩法**：学单词 · 听力发音 · 拼一拼 · 闯关测验 · 翻牌配对 · 打地鼠
- **🆕 情景对话**：8 段生活对话（打招呼/介绍/家庭/颜色/课堂/食物/时间/告别），听 + 选答句
- **跟读评分**：Web Speech API 录音 → 0-3 分评级（依赖浏览器麦克风权限）
- **发音修正**：Mr / Mrs / Ms / T-shirt / o'clock 等正确读法表

### 🔢 数学乐学（苏教版）
- **24 章节全覆盖**：3A 13 单元 + 3B 11 单元
- **13 题型生成器**：口算·乘法口诀·单位换算·应用题·混合运算·24时·两/三位×一位·两位×两位·除法·重量·长度·年月日·长方形周长面积·分数·角度
- **🆕 看图分数**：随机网格涂色 → 选分数表达
- **🆕 审题"圈关键词"**：应用题做题前先点出数字 + 问题词，分离"读题"和"计算"
- **自适应难度**：每次会话 _sessionDiff 从 2 起，连对 3 升 3，连错 2 降 1

### 📖 语文乐学（统编版）
- **41 课全收 · 501 字写字表**：3A 20 课 + 3B 21 课，每课带生字+拼音+词组
- **14 首课内古诗**：山行/赠刘景文/夜书所见/望天门山/...
- **6 玩法**：生字认读 · 拼音练习 · 古诗背诵 · 词语听写 · 🆕 阅读理解 · 🆕 错题专项
- **🆕 笔顺动画**：每个生字点「✍️ 看笔顺」，hanzi-writer 动画演示书写顺序
- **🆕 阅读理解**：6 篇精选短文 + 每篇 3 道题（信息提取 / 词义 / 主旨）

### 🎒 学生主页（Hub）
- **今天总览**：分钟 / 题数 / 平均正确率
- **🎯 今日任务**：3 张智能派单卡（复习到期错题 / 新学未开科 / 弱项强化）
- **🏅 成就勋章 12 枚**：🌱🐝⭐👑🔥💪🏆📚📕🎯🌟💯
- **📈 本周三科趋势**：7 × 3 柱状图，看哪天偏科
- **📸 导出 PNG**：一键截图周报，转发家长群

### 📊 家长后台（parent.html）— Wave 3 新增
- **🔒 4 位口令门禁**（默认 `0528`，可改）+ 半小时 session 自动锁
- **📈 综合 KPI**：近 7 日分钟 / 题数 / 正确率 / 错题本
- **🗓️ 30 天热力图**：5 色梯度，断更一目了然
- **🥧 错因分布饼图**：知道孩子错在「字形混淆/口诀/审题/...」
- **📚 三科 7 天正确率对比**：哪科是短板
- **🎯 Top 5 高频错题**：错答最多的题，含 errorType 标注
- **🎯 本周目标 + 进度条**：家长设"每天 15 分钟 × 5 天"，孩子打卡
- **🚨 智能提示 3 档**：⚠️ 断更/正确率下滑/错题暴增 · 💡 可复习 · 🌟 表扬
- **🛠️ 工具栏**：导出 PNG 周报 / 打印 A4 / 导出 JSON / 一键清空

## 🛠️ 技术栈

| 层 | 选型 | 理由 |
|---|---|---|
| 前端 | 单文件 HTML + 原生 JS | 零构建，单文件 60KB 内，离线可用 |
| 存储 | localStorage（按 `en_` / `m_` / `c_` 前缀分科） | 同源共享，零后端 |
| TTS | Web Speech API | 浏览器原生，无费用 |
| 跟读评分 | webkitSpeechRecognition + Levenshtein 距离 | 0-3 分粒度 |
| 音效 | Web Audio API 现场合成 | 零音频文件 |
| 笔顺 | [hanzi-writer 3.5](https://hanziwriter.org) CDN | 8000+ 汉字数据库 |
| 截图 | [html2canvas 1.4](https://html2canvas.hertzen.com/) CDN | DOM → PNG |
| 部署 | GitHub Pages（4 个独立 repo） | 同源 = 共享 localStorage |

## 🚀 部署流程

每个模块都是独立 GitHub repo，单文件 `index.html`：

```
TideOneZPP/enapp        → tideonezpp.github.io/enapp/
TideOneZPP/mathapp      → tideonezpp.github.io/mathapp/
TideOneZPP/chineseapp   → tideonezpp.github.io/chineseapp/
TideOneZPP/hubapp       → tideonezpp.github.io/hubapp/  (+ parent.html)
```

修改 → `git commit` → `git push origin main` → GitHub Pages 约 30-90 秒后生效。
