# 📘 乐学使用说明

> 写给两类人：**家长**（怎么帮孩子用好这套系统）和 **孩子**（每天怎么学）。

---

# 第一部分 · 给家长

## 一、首次设置（5 分钟）

### 1. 把 App 装到孩子的 iPad

1. 用 Safari 打开 https://tideonezpp.github.io/hubapp/
2. 点底部 **分享按钮** → **添加到主屏幕**
3. 命名为「乐学」→ 添加
4. 主屏幕会出现 🎒 图标，全屏打开，跟原生 App 一样

> ⚠️ 推荐用 **Safari**，因为：① TTS 中英文都有 ② 麦克风权限弹窗最规范 ③ html2canvas 截图最快

### 2. 改家长后台口令

1. 在 iPad 任何浏览器（不要让孩子看到）打开：https://tideonezpp.github.io/hubapp/parent.html
2. 输入默认口令 **0528**（孩子生日月日）
3. 进入后点右上角 **⚙️ 改口令**
4. 输入你自己设的 4 位数字（别用 0000/1234 这种好猜的）

> 💡 口令存在浏览器 localStorage，**只对你这个浏览器有效**。换浏览器需重新设。

### 3. 设定本周目标

1. 进入家长后台
2. 在「🎯 本周目标」处输入：
   - **每天目标分钟数**：三年级建议 15-20 分钟（番茄钟一个周期）
   - **本周达标几天**：周一到周五 = 5 天比较合适，周末让孩子放松
3. 点「保存」

## 二、日常陪伴节奏

### 学习节奏建议

```
周一-周五：每天 15 分钟，三科轮换
  - 周一/周四：英语 + 复习语文错题
  - 周二/周五：数学 + 复习英语错词
  - 周三：语文 + 复习数学错题
周六：自由选 1 小时，挑战进阶题
周日：休息 / 看孩子主动复习
```

> 关键不是每天学多久，是**每天都打开**（连击勋章 🔥 在等他）。

### 每天看什么

打开家长后台（半小时 session 内不用再输口令），从上到下扫一眼：

| 看什么 | 看什么信号 | 该做什么 |
|---|---|---|
| **综合 KPI** | 7 日总分钟 < 60 | 不够。30 分钟陪学，找回节奏 |
| | 正确率 < 70% | 题太难或不熟，去错题专项 |
| **30 天热力图** | 出现连续 2-3 个灰格 | 孩子断更，问问怎么了 |
| **错因饼图** | "字形混淆"占比 > 50% | 写字本翻出来，每天写 5 个 |
| | "口诀"占比高 | 乘法口诀表贴墙上 |
| **三科条形** | 某科明显短 | 那科多陪 5 分钟 |
| **Top 5 高频错题** | 都是同一类型 | 拿来当饭桌话题 |
| **智能提示** | ⚠️ 出现 | 当天就处理 |

### 每周做什么

**周日晚上 10 分钟仪式感**：

1. 打开家长后台
2. 截图 PNG 周报（按钮在最底下）→ 发给孩子和爷爷奶奶 / 班级群
3. 跟孩子一起看：
   - 「这周你坚持了 X 天」（解锁 🔥 给他看）
   - 「你最常错的是 ___，咱们下周怎么搞定它？」
   - 「下周目标改一下：从 X 天 → Y 天」

## 三、常见疑问

### Q: 孩子能进家长后台吗？
- 不能。4 位口令拦住。半小时 session 一过自动锁。
- 如果忘了口令，**清浏览器 localStorage 的 `parent_pin` 字段**即可重置默认 0528。

### Q: 换 iPad 怎么把进度带过去？
- 家长后台底部点 **💾 导出全部数据 (JSON)** → 保存文件
- 新 iPad 打开任一 App 的浏览器 console（开发者菜单）
- 粘贴：`Object.entries(导出的JSON).forEach(([k,v])=>localStorage.setItem(k,v))`
- 刷新页面即可

> 后续版本会加 iCloud 同步，省去这一步。

### Q: 孩子按了"清空学习记录"会怎样？
- 学生 App 内的清空按钮**必须输入"确认清空"4 个字**才生效。
- 这是为了防止误点。三年级孩子不会顺手输入这 4 个字。

### Q: 家长后台显示的"30 天"是哪 30 天？
- 是滚动窗口：今天往前数 30 天。每天会自动滑动。

### Q: 错题为什么有的还没消化就不弹了？
- 这是**艾宾浩斯遗忘曲线**在工作。错题答对 1 次 → 1 天后再考；再答对 → 2 天后；4 天 → 7 天 → 15 天 → 30 天。
- **5 次间隔答对 = 真正掌握**，自动从错题本毕业。
- 没毕业前每次到期都会进 30% 复习池。

### Q: 番茄钟到 25 分钟强制结束了，孩子哭闹？
- 这是**保护视力 + 培养节奏**。说"明天再来 25 分钟"，他第二天会更主动打开。
- 如果要改：在每个 App 源码里搜 `if(min>=25)` 改阈值。但教育学不建议放宽。

### Q: 数据丢了怎么办？
- localStorage 只在本地浏览器。**不要清浏览器缓存**，不要"还原 iPad"。
- 推荐每周日导出一次 JSON 备份到 iCloud Drive。

# 第二部分 · 给孩子

## 一、🎒 进入「乐学」

主屏幕上点 🎒 图标 → 看到三科卡片 → 想学哪科点哪科。

不知道学什么？看主页中间的 **🎯 今日任务** 卡片，**听 App 派单**就好：
- 📕 卡片 = 复习以前错过的题（最重要！）
- 🆕 卡片 = 新学一组（今天还没打开的科目）
- 🎯 卡片 = 弱项强化（今天正确率最低的科目）

## 二、🇬🇧 学英语

| 玩法 | 怎么玩 |
|---|---|
| 📚 **学单词** | 看图认词，点 🔊 听发音，点 🎤 自己读，App 给你打分 |
| 👂 **听力发音** | 听一段，从 4 个里选对的 |
| 🔤 **拼一拼** | 看中文，自己拼字母 |
| 🏆 **闯关测验** | 混合题，连击越多奖励越大 |
| 🃏 **翻牌配对** | 图 ↔ 词，练记忆 |
| 🔨 **打地鼠** | 听音拍正确的词，限时挑战 |
| 💬 **情景对话** ⭐NEW | 听对话，选最合适的回答 |

**💡 小窍门**：错的词会进**错词本**。每天打开 App，先把今日任务的"复习"做完，**连击 🔥** 蹿得最快。

## 三、🔢 学数学

主页选**「按课本目录」**→ 选当前在学的章（比如 3A 第三章「乘一位数」）→ 直接做题。

应用题做之前 App 会让你 **👆 圈出关键词**：
- 圈数字（多少 / 几 / 一共）
- 圈问题（求什么？）

**圈完才能答题**。这是审题训练，把"看错题"和"算错题"分开。

错了不要怕，App 会出现：
- 💡 解释卡：告诉你为什么错
- 🏷️ 错因 chips：你**自己点**「口算 / 口诀 / 粗心」其中一个 — 这是教你学会"反思错在哪"

## 六、🍅 番茄钟规则

打开 App 学习时，**屏幕顶部**会出现一个小番茄计时：

- `🍅 0/15 min` → 刚开始
- `🍅 15/15 min` → 颜色变橙 = "可以休息一下了"
- `⏰ 该结束了 (25min)` → 红色 = 强制结束，明天再来！

> 这不是逼你，是**保护眼睛**。番茄一响就停 — 你会发现明天会更想学。

## 七、😅 错了不要怕

错题 = 你的私人补给包。**每个错题平均 5 次见面就掌握**：

```
第 1 次错        → 立刻可复习
第 2 次（答对）  → 1 天后再考
第 3 次（答对）  → 2 天后再考
第 4 次（答对）  → 4 天后再考
第 5 次（答对）  → 7 天后再考
…
答对 5 次 → 🎓 毕业！从错题本消失
```

App 会**自动到期就弹出来**，你只管做。

# 第三部分 · 故障排查

| 现象 | 原因 | 怎么办 |
|---|---|---|
| 第一次点 🔊 没声音 | iOS 安全策略：需用户先互动 | 再点一次 |
| Mr / o'clock 读错 | 已修正，刷新页面试试 | 检查 SPEAK_FIX 表是否生效 |
| 笔顺动画转圈 | hanzi-writer CDN 网络慢 | 等几秒，或刷新；个别冷门字可能无数据 |
| 打字时弹键盘挡题目 | iPad Safari 默认行为 | 把 iPad 横屏 |
| 跟读评分总是 0 分 | 麦克风权限被拒 | 设置 → Safari → 麦克风 → 允许 |
| 截图 PNG 一片空白 | 第三方库（html2canvas）的 emoji 兼容问题 | 多按一次「📸 导出」；或用 iPad 系统截屏代替 |
| 家长后台进不去 | 口令忘了 | 清浏览器 localStorage 的 `parent_pin` 字段 |
| 数据全没了 | 浏览器缓存被清/换 iPad | 之前导出过 JSON 吗？有的话恢复；没有就重新开始 |
| 番茄钟一直显示 0 min | 已经 home() 退出了，再进 drill 才会启动 | 进入任一题目模式即可见 |

# 第四部分 · 隐私与安全

- **所有数据存在 iPad 本地**（localStorage），**不上传任何服务器**
- 我们不知道你孩子叫什么、几岁、答对了多少
- 4 位口令防孩子误闯家长后台，半小时 session 自动失效
- "清空学习记录"按钮**必须输入"确认清空"**才生效，防止误点
- 家长后台清空按钮要求输入"全部清空"才生效
